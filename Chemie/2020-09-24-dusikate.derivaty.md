# Dusíkaté deriváty

1. Aminy - NH~2~
2. Nitrosloučeníny - NO~2~

## Aminy

- jedovaté
- karcinogenní
- léky

```
	_
H - N - H
	|
	H
```

1. **Primární** - nahrazen 1H

   CH3 NH2

   ```
   	 _
   R1 - N - H
   	 |
   	 H
   ```

2. **Sekundární** - nahrazeny 2H

   ```
   	 _
   R1 - N - R2
   	 |
   	 H
   ```

   CH3 - NH - C2H3  (ethylmethylamin)

3. **Terciální** - nahrazeny 3H

   ```
   	 _
   R1 - N - R2
   	 |
   	 R3
   ```

**monoamin** = NH2 CH2 CH3

**diaminy**  = **NH2**CH2CH2**NH2**
